# FSEC-personal-portfolio
2025年东南大学方程式赛车整车控制集成板：包含DCDC电源供电、安全线路、故障检测与切断、信号灯控制等功能  
  
电源：24V DCDC  
连续串联开关（车辆安全项）对动力电池主继电器线圈供电  
检测BMS（电池管理系统）、IMD（绝缘检测）并做电路故障切断  
信号灯的可视性检查，刹车灯控制，TSAL（驱动系统状态逻辑）等
