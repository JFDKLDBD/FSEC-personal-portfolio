# FSEC-personal-portfolio
两个文件注释：  

ESF是赛车电气原理及安全系统检查表，需从原理图到细节完全展示车辆的设计合理性和选材细节。2024年是疫情结束、比赛结束停摆后，第一次审核尺度的大幅提升，这份ESF是由我完全主导编写的车辆电气原理及安全细节，囊括驱动（高压）系统的原理图及细节、动力电池的原理图及细节、安全（切断）回路的原理图及细节，故障锁存、状态逻辑的原理图及细节。同时需要事无巨细计整车样材、线缆、绝缘材料等位置及使用。其审查范围之广度、深度、细节度都是对于车规级安全的重要考验。  

SES是车辆机械结构方面的同等性证明，车队需要展示设计参数是否合规，对车辆的不同结构设计、材料及形状选择，通过各种实验或者计算，证明其相对于模板安全结构的安全和设计冗余。
